# 🪶 Inspira Ink

**Inspira Ink** is a premium, minimalist web application designed as a sanctuary for writers and dreamers. It provides a clean, aesthetic space to explore thousands of quotes categorized by emotion and immortalize your own lines in a permanent digital archive.

## ✨ Live Demo
Experience the ink at: **[https://inspira-ink.vercel.app](https://inspira-ink.vercel.app)**

## 🖋️ Key Features

- **Inky Minimalist Aesthetic**: A deep, dark blue "Inky" theme designed for focus and elegance.
- **7 Emotional Realms**: Navigate through 14,049 pre-loaded quotes across 7 distinct moods:
  - Happy, Sad, Inspired, Angry, Calm, Anxious, and Romantic.
- **Immortalize Your Lines**: Writers can submit their own quotes, which are checked for basic grammar and meaningfulness before being saved.
- **Infinite Persistence**: Built with a cloud-connected KV database to ensure your quotes are stored permanently—not just for your session, but for the world to see forever.
- **Search by Inscription ID**: Every quote is assigned a unique number (starting from #14050 for new users). Use the search bar to find and share specific inscriptions.
- **Clean UI/UX**: Powered by Next.js and Lucide Icons (featuring the signature Feather).

## 🛠️ Technology Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: Vanilla CSS (Custom Design System)
- **Database**: 
  - Local: SQLite (Seeding)
  - Production: Cloud KV Store (Infinite Persistence)
- **Deployment**: [Vercel](https://vercel.com/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/LeoxIrfzzzz/Inspira-Ink.git
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Run the development server**:
   ```bash
   npm run dev
   ```
4. **Open in browser**:
   Navigate to [http://localhost:3000](http://localhost:3000)
